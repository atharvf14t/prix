# **App Name**: Eternal Flame

## Core Features:

- Valentine Card Display: Displays a central card with the message: 'Naisha, will you be my valentine?'
- Interactive 'No' Button: The 'No' button dynamically moves away from touch/cursor, making it unclickable.
- Affirmative Email: On 'Yes' click, an email is sent to atharvdxb14@gmail.com with the subject 'Valentine day- Naisha Agreed!!!'
- Celebratory Animation: Triggers a sparkle animation and displays 'Good girl' for 2 seconds with a themed background.
- Email Response Tool: An AI tool reviews email for matches based on prior email messages.

## Style Guidelines:

- Primary color: Deep rose (#C72375) for romance and passion.
- Background color: Light pink (#F8D6E8) for a soft, loving backdrop.
- Accent color: Burgundy (#800020) for buttons and interactive elements.
- Font: 'Belleza' for headlines, lending a touch of art and fashion to the text; for body text use 'Alegreya', a humanist serif with an elegant and intellectual feel.
- Centered card layout to focus the user's attention on the valentine proposal.
- Sparkle animations and a smooth transition for the celebratory message.
